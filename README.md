## 💻 What I Build

### 🎵 Featured Project: [Sing Unto The Lord](https://github.com/KizioTech/sing-unto-the-lord)

<div align="center">

![Version](https://img.shields.io/badge/version-1.7.0-purple)
![React](https://img.shields.io/badge/React-18-61DAFB?logo=react&logoColor=white)
![Tailwind CSS](https://img.shields.io/badge/Tailwind%20CSS-%2338B2AC?logo=tailwindcss&logoColor=white)
![PWA](https://img.shields.io/badge/PWA-Enabled-0052cc)

</div>

A modern, responsive hymnbook application designed for seamless worship at the **University of Malawi Church of Christ**. This progressive web app brings timeless hymns into the digital age with an elegant interface and powerful features.

**Key Features:**
- 🎼 Comprehensive hymn database with advanced search & filtering
- ⭐ Favorites system and recently viewed history
- 🌙 Dark/Light theme with adjustable font sizes
- 📱 Full PWA support for offline access
- 🖥️ Presentation mode with auto-scroll
- 🎬 YouTube integration for hymn videos
- ⌨️ Keyboard shortcuts for efficient navigation
- 📐 Responsive design for all devices

**Tech Stack:** React 18 • Tailwind CSS • Service Workers • Lucide Icons • PWA
